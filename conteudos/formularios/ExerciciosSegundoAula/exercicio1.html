Elementos de formulários HTML (INPUT | SELECT | TEXTAREA)

INPUT

Propósito: a tag INPUT define um campo de entrada de dados. Em um formulário, cada campo é considerado uma variável que possui um nome e um tipo específico.

Utilização:

<input type="text" name="nome" value="Gustavo Rissetti" size=8 maxlength=20>



<input type="password" name="senha" value="12345" size=6 maxlength=6>



<input type="radio" name="sexo" value="M"> Masculino <br>
<input type="radio" name="sexo" value="F"> Feminino

Masculino
Feminino

<input type="checkbox" name="ida_e_volta" value="S" checked> Ida e volta

 Ida e volta
<input type="hidden" name="escondido" value="este valor estava escondido">

<input type="file" size="90" name="curriculo">

<input type="button" name="ok" value="ok" onClick="alert('ok');" >



<input type="reset" name="limpar" value="Limpar" >



<input type="submit" name="enviar" value="Enviar" >



<input type="image" src="figura.jpg" value="submit" >
Parâmetros:
  • type: define o tipo de entrada, podendo ser:
    • text: área de texto;
    • password: texto com leitura protegida (campo senha);
    • hidden: entrada oculta, não possui visualização;
    • radio: botão de opção única;
    • checkbox: botão de opção múltipla;
    • button: botão com ação a ser definida pelo programador;
    • reset: botão com ação de limpar os dados do formulário;
    • submit: botão com ação de enviar (submeter) os dados para a ação do formulário.
    • image: imagem clicável para submeter os dados para a ação do formulário.

  • name: define o nome do campo (nome do objeto para o formulário)
  • value: define o valor do campo
  • checked: indica que o campo está ou não marcado (utilizado para valores padrão)
  • size: especifica o tamanho do campo (a quantidade de caracteres a serem exibidos)
  • maxlength: define a quantidade máxima de caracteres que o campo poderá receber.
  • src: se o tipo do input for image o parâmetro src indica o caminho do arquivo de imagem.
SELECT

Propósito: a tag SELECT define uma lista de itens que podem ser selecionados pelo usuário.

Utilização:

<select name="sexo" size=2>
<option>Masculino </option>
<option>Feminino </option>
</select>



<select name="UF" size=1>
<option>RS </option>
<option>SC </option>
<option>PR </option>
</select>



<select name="cursos" size=3 multiple>
<option>Técnico em Informática </option>
<option>Análise e Desenvolvimento de Sistemas </option>
<option>MSI </option>
</select>



Parâmetros:
  • name: define o nome do campo (nome do objeto para o formulário)
  • size: define o número de itens que serão exibidos (se o tamanho for igual a um então o campo se comporta como uma caixa de seleção).
  • multiple: define que vários itens da lista poderão ser selecionados.
  • option: define as opções da lista, que deverão ser delimitadas pelas tags <option> <\option>.
TEXTAREA

Propósito: a tag TEXTAREA define uma área de texto com múltiplas linhas.

Utilização:

<textarea name="memorial" rows=4 cols=50>
Aqui vai o texto...
<textarea>

Parâmetros:
  • name: define o nome do campo (nome do objeto para o formulário)
  • rows: define o número de linhas da área de texto.
  • cols: define o número de colunas da área de texto.
